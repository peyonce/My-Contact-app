 # React Native Chat UI

A clean and modern chat interface built with React Native and Expo for the React Native UI Challenge.

## What I Built

**Actual Implementation:**
- Main chat list screen with recent conversations
- Contact avatars with user initials  
- Chat items with user names, timestamps, and message previews
- Typing indicators and voice message status
- Group chat sections
- Responsive design for different screen sizes
- Reusable `ChatItem` and `ContactAvatar` components

## Technology Used

- React Native with Expo
- TypeScript
- React Native StyleSheet

##  How to Run

```bash
npm install
npm start
